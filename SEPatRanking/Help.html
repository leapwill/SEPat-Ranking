Main Window

This window is shown when the program starts. It provides the ability to search, create, and modify student information. It also provides links to all other parts of the program.

Searching Students

First, select the category you want to search by. Last name, first name, and ID number are available. After selecting, type your search terms in the "Query" box and press enter or the "Search" button to show results in the view at the bottom of the window. Click the "New Window" button to open the Edit Database window with your search results.
